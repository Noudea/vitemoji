# vitemoji

`vitemoji` is a small TypeScript workspace for experimenting with UI text transforms inside Vite.

## Workspace

- `packages/vitemoji`: the plugin package
- `playground`: a local React + Vite app

## Current Scope

- Transform JSX text like `hello world` into `👋 🌍`
- Transform JSX text like `fire` into `🔥`
- Stay in standard TypeScript and standard TSX

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run typecheck`
