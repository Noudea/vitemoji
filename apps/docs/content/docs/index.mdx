`vitemoji` is a Vite plugin and text utility that rewrites UI text into emoji.

It parses `.jsx` and `.tsx` modules, finds text rendered in JSX, and replaces
matching values using `emojibase-data`.

## What It Does

The plugin rewrites:

- JSX text like ``<h1>hello world</h1>``
- string literals rendered inside JSX like ``<p>{"hello world"}</p>``
- template literal text rendered inside JSX like ``<p>{`smile ${name}`}</p>``

The plugin does not rewrite:

- files in `node_modules`
- files outside the configured `include` pattern
- plain strings outside JSX
- code that fails to parse as TypeScript or JSX

## Quick Example

```tsx
export function App() {
  const name = 'builder';

  return (
    <main>
      <h1>hello world</h1>
      <p>{":fire:"}</p>
      <p>{`smile ${name}`}</p>
    </main>
  );
}
```

With `preset: "chaos"`, this can become:

```tsx
export function App() {
  const name = 'builder';

  return (
    <main>
      <h1>👋 🌍</h1>
      <p>{"🔥"}</p>
      <p>{`😀 ${name}`}</p>
    </main>
  );
}
```

## Install From npm

```bash
pnpm add vitemoji
```

Then add the plugin to your Vite config:

```ts
import react from "@vitejs/plugin-react";
import { defineConfig } from "vite";
import { vitemoji } from "vitemoji";

export default defineConfig({
  plugins: [
    react(),
    vitemoji(),
  ],
});
```

You can also use the plain text utility without the plugin:

```ts
import { emojifyText } from "vitemoji";

emojifyText(":fire:");
// "🔥"
```
